# Contributing

Thanks for wanting to contribute! Here's how to get started.

## Code of Conduct

Be respectful and constructive. We're all here to build something great together.

- Be welcoming and inclusive
- Accept constructive feedback gracefully
- No harassment, trolling, or personal attacks
- If a disagreement arises, discuss it calmly in the relevant issue or PR â€” escalate to a maintainer if needed

## How to Contribute

1. Fork the repo and create a branch
2. Make your changes
3. Open a pull request using the PR template
4. A maintainer will review and merge

For bugs or feature ideas, [open an issue](../../issues/new/choose) using one of the provided templates.

## Pull Request Guidelines

- Keep changes focused and well-described
- Reference any related issues
- Make sure your branch is up to date with `main`

Thanks for your help!
